[ PHASE 1: DISCOVERY ]

THEORY: VERBOSE ERRORS

Developers often leave debug mode enabled in production. When an error occurs, the server might return a full stack trace or even environment variables.

OBJECTIVE

Trigger an error and find hidden secrets in the debug output.

HINT 1: TRIGGERING ERRORS

The "System Diagnostics" panel has a button "RUN SYSTEM CHECK". This button seems to fail intentionally.

Click it and carefully read the output.

HINT 2: ENVIRONMENT VARIABLES

Look for a section labeled DEBUG CONTEXT or similar. In Node.js, process.env contains sensitive keys.

FULL EXECUTION PLAN

  1. Click "RUN SYSTEM CHECK".
  2. Scroll down to the yellow text section.
  3. Look for keys like API_KEY or SECRET inside the JSON dump.
