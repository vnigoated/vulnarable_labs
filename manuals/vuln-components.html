[ PHASE 1: VULNERABILITY IDENTIFICATION ]

THEORY: RCE via EVAL

eval() allows executing arbitrary code. If user input is passed to it, the user can run any command the server process can.

OBJECTIVE

Execute file system commands to list the directory contents.

HINT 1: JAVASCRIPT EXECUTION

Since the backend is Node.js, you can run JavaScript code.

Try entering global.process.version to see if it returns the Node version.

HINT 2: MODULE ACCESS

Node.js has a module child_process that can run OS commands.

You can require it directly if the sandbox is weak: require('child_process').

FULL EXECUTION PLAN

Enter the following payload into the calculator:

require('child_process').execSync('dir').toString()

(Use ls instead of dir if on Linux/Mac).

The result will show the file listing of the server.
